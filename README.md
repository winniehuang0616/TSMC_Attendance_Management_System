# TSMA ATTENDANCE MANAGEMENT SYSTEM

### 執行方式
1. cd frontend
2. yarn ( 裝套件 )
3. yarn dev ( 跑起來 )

### 開發流程
1. git pull (in main)
2. git checkout -b "component name"
3. coding...........
4. yarn lint && yarn format 
5. git add .
6. git commit -m "格式參照文件"
7. git rebase main ( 解衝突 )
8. git push
9. 到 github 上檢查是否通過測試，通過沒問題後可合併
